# Underscore — Voice Assistant 🎙️

A desktop voice assistant built with Python

## Features
- Wake word detection ("Underscore")
- Open websites and apps by voice
- Wikipedia summaries
- Date and time
- Real-time GUI with status indicator
- Plugin-based command system
- Full logging to file

## Tech Stack
- Python
- SpeechRecognition
- pyttsx3
- tkinter
- threading

## Setup
pip install -r requirements.txt
python main.py

## How To Use
1. Click START
2. Say "Underscore" to wake it up
3. Give a command like "open youtube" or "tell me about Einstein"
4. Say "stop" to deactivate
